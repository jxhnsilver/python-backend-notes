### –®–∞–±–ª–æ–Ω 5. –ì–µ–Ω–µ—Ä–∞—Ü–∏—è –º–∞—Ç—Ä–∏—Ü—ã

```python
def generate_int_matrix(rows: int, cols: int) -> list[list[int]]:
    matrix = [list(range(i * cols, (i + 1) * cols)) for i in range(rows)]
    return matrix
```

## –ö–æ–≥–¥–∞ –ù–ï –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `range`

- –î–ª—è **–≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã—Ö —á–∏—Å–µ–ª** ‚Üí –ø–∏—à–∏ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä:
```python
def frange(start, stop, step):
    while start < stop:
        yield start
        start += step
```
- –î–ª—è **–Ω–µ–∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–µ–π** ‚Üí —Å–ø–∏—Å–∫–∏, –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä—ã.
- –î–ª—è **—Ä–∞–±–æ—Ç—ã —Å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –Ω–∞–ø—Ä—è–º—É—é** ‚Üí `for item in collection`.
## –ò—Ç–æ–≥: –ª—É—á—à–∏–µ –ø—Ä–∞–∫—Ç–∏–∫–∏

1. **–ò—Å–ø–æ–ª—å–∑—É–π `range` –¥–ª—è –∏–Ω–¥–µ–∫—Å–æ–≤ –∏ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–µ–π**.
2. **–ù–µ –æ–±–æ—Ä–∞—á–∏–≤–∞–π –≤ `list` –±–µ–∑ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏** ‚Äî —Ç–µ—Ä—è–µ—à—å —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å.
3. **–ü—Ä–æ–≤–µ—Ä—è–π –≤—Ö–æ–∂–¥–µ–Ω–∏–µ —á–µ—Ä–µ–∑ `in` ‚Äî —ç—Ç–æ O(1)!**
4. **–ò–∑–±–µ–≥–∞–π `range(len(...))`, –µ—Å–ª–∏ –º–æ–∂–Ω–æ –∏—Ç–µ—Ä–∏—Ä–æ–≤–∞—Ç—å—Å—è –Ω–∞–ø—Ä—è–º—É—é**:
```python

    print(items[i])

    print(item)
```
