### –û—Å–Ω–æ–≤–Ω—ã–µ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏ `list`

| --------------------- | -------------------------------------------------------------------------------------------------------------- |

### –û—Å–Ω–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –∏ –º–µ—Ç–æ–¥—ã

### 1. `list.append(x)`
**–î–æ–±–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç `x` –≤ –∫–æ–Ω–µ—Ü —Å–ø–∏—Å–∫–∞.**
```python

```
### 2. `list.extend(iterable)`
**–î–æ–±–∞–≤–ª—è–µ—Ç –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –¥—Ä—É–≥–æ–≥–æ —Å–ø–∏—Å–∫–∞) –≤ –∫–æ–Ω–µ—Ü.**
```python

```
‚ö†Ô∏è –ù–µ –ø—É—Ç–∞–π —Å `append`: `a.append([3,4])` –¥–æ–±–∞–≤–∏—Ç **–≤–µ—Å—å —Å–ø–∏—Å–æ–∫ –∫–∞–∫ –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç** ‚Üí `[1, 2, [3, 4]]`.
### 3. `list.pop([index])`
**–£–¥–∞–ª—è–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –ø–æ –∏–Ω–¥–µ–∫—Å—É (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é ‚Äî –ø–æ—Å–ª–µ–¥–Ω–∏–π).**
```python

```
### 4. `list.remove(value)`

**–°—á–∏—Ç–∞–µ—Ç, —Å–∫–æ–ª—å–∫–æ —Ä–∞–∑ `value` –≤—Å—Ç—Ä–µ—á–∞–µ—Ç—Å—è –≤ —Å–ø–∏—Å–∫–µ.**

### 7. `list.sort(key=None, reverse=False)`
**–°–æ—Ä—Ç–∏—Ä—É–µ—Ç —Å–ø–∏—Å–æ–∫ **–Ω–∞ –º–µ—Å—Ç–µ** (–º–µ–Ω—è–µ—Ç –∏—Å—Ö–æ–¥–Ω—ã–π —Å–ø–∏—Å–æ–∫).**

| –£–Ω–∏–∫–∞–ª—å–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è (–µ—Å–ª–∏ –ø–æ—Ä—è–¥–æ–∫ –Ω–µ –≤–∞–∂–µ–Ω) | `list(set(items))`            |

### –ö–∞–∫ —Ö—Ä–∞–Ω–∏—Ç—Å—è –≤ –ø–∞–º—è—Ç–∏

- –í–Ω—É—Ç—Ä–∏ `list` ‚Äî –º–∞—Å—Å–∏–≤ **—É–∫–∞–∑–∞—Ç–µ–ª–µ–π** (`PyObject*`) –Ω–∞ –æ–±—ä–µ–∫—Ç—ã –≤ –∫—É—á–µ.
- –ü—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏ —ç–ª–µ–º–µ–Ω—Ç–∞, –µ—Å–ª–∏ –ø–∞–º—è—Ç—å –∏—Å—á–µ—Ä–ø–∞–Ω–∞ ‚Äî –≤—ã–¥–µ–ª—è–µ—Ç—Å—è **–Ω–æ–≤—ã–π –±–ª–æ–∫** (~1.125√ó –±–æ–ª—å—à–µ), —Å—Ç–∞—Ä—ã–µ —É–∫–∞–∑–∞—Ç–µ–ª–∏ –∫–æ–ø–∏—Ä—É—é—Ç—Å—è —Ç—É–¥–∞.
- –û–±—ä–µ–∫—Ç—ã **–Ω–µ —Ö—Ä–∞–Ω—è—Ç—Å—è –≤–Ω—É—Ç—Ä–∏ —Å–ø–∏—Å–∫–∞**, —Ç–æ–ª—å–∫–æ —Å—Å—ã–ª–∫–∏ –Ω–∞ –Ω–∏—Ö.

```python

# a ‚Äî —Å–ø–∏—Å–æ–∫ –∏–∑ 3 —É–∫–∞–∑–∞—Ç–µ–ª–µ–π:
# ‚Üí int(1), str("hello"), list([])
```

### –ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `list`

- –î–ª—è **—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**
- –ö–æ–≥–¥–∞ –Ω—É–∂–µ–Ω **–±—ã—Å—Ç—Ä—ã–π –¥–æ—Å—Ç—É–ø –ø–æ –∏–Ω–¥–µ–∫—Å—É**
- –ö–æ–≥–¥–∞ —á–∞—Å—Ç–æ **–¥–æ–±–∞–≤–ª—è–µ—Ç–µ/—É–¥–∞–ª—è–µ—Ç–µ —Å –∫–æ–Ω—Ü–∞**
- –ö–æ–≥–¥–∞ **–Ω–µ –Ω—É–∂–Ω–∞ –≥–∞—Ä–∞–Ω—Ç–∏—è —É–Ω–∏–∫–∞–ª—å–Ω–æ—Å—Ç–∏**

> ‚ö†Ô∏è **–ù–µ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ `list` –∫–∞–∫ –æ—á–µ—Ä–µ–¥—å** (`pop(0)` ‚Üí O(n)). –î–ª—è –æ—á–µ—Ä–µ–¥–∏ ‚Äî `collections.deque`

### –ü–æ—á–µ–º—É —ç—Ç–æ "–ª—É—á—à–µ"

- **–ü—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å**: `append`/`pop` —Å –∫–æ–Ω—Ü–∞ ‚Äî O(1), —á—Ç–æ –∫—Ä–∏—Ç–∏—á–Ω–æ –¥–ª—è —Å—Ç–µ–∫–æ–≤ –∏ –±—É—Ñ–µ—Ä–æ–≤.
- **–ß–∏—Ç–∞–µ–º–æ—Å—Ç—å**: –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å, –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏.
- **–ì–∏–±–∫–æ—Å—Ç—å**: –ø–æ–¥–¥–µ—Ä–∂–∫–∞ —Å—Ä–µ–∑–æ–≤, –∏—Ç–µ—Ä–∞—Ü–∏–π, comprehensions.
- **–°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–µ SOLID**: –∏–Ω–∫–∞–ø—Å—É–ª–∏—Ä—É–µ—Ç –ø–æ–≤–µ–¥–µ–Ω–∏–µ –∫–æ–ª–ª–µ–∫—Ü–∏–∏, –ª–µ–≥–∫–æ —Ç–µ—Å—Ç–∏—Ä—É–µ–º, –ø–æ–¥–º–µ–Ω—è–µ–º (–Ω–∞–ø—Ä–∏–º–µ—Ä, –Ω–∞ `tuple` –¥–ª—è –Ω–µ–∏–∑–º–µ–Ω—è–µ–º–æ—Å—Ç–∏).

–ï—Å–ª–∏ –Ω—É–∂–Ω–∞ **–æ—á–µ—Ä–µ–¥—å (FIFO)** ‚Äî –∏—Å–ø–æ–ª—å–∑—É–π `collections.deque`.  
–ï—Å–ª–∏ –Ω—É–∂–µ–Ω **—Å—Ç–µ–∫ (LIFO)** ‚Äî `list` –∏–¥–µ–∞–ª–µ–Ω (`append` / `pop`).
