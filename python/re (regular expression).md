## üõ† –ü—Ä–∏–º–µ—Ä—ã —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Ç–µ–∫—Å—Ç–∞

### 1. –£–¥–∞–ª–µ–Ω–∏–µ –ª–∏—à–Ω–∏—Ö –ø—Ä–æ–±–µ–ª–æ–≤
```python
import re

```
### 2. –ò–∑–≤–ª–µ—á–µ–Ω–∏–µ email-–∞–¥—Ä–µ—Å–æ–≤
```python
text = "–°–≤—è–∂–∏—Ç–µ—Å—å —Å –Ω–∞–º–∏: admin@example.com –∏–ª–∏ support@test.org"
emails = re.findall(r'\b[\w.-]+@[\w.-]+\.\w+\b', text)

```
### 3. –§–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —Ç–µ–ª–µ—Ñ–æ–Ω–æ–≤

```python
phone = "+7 (999) 123-45-67"

```
