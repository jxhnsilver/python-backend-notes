### **DATABASES в settings.py**
```python
# Ваш текущий файл django_learning/settings.py
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',  # Движок базы
        'NAME': BASE_DIR / 'db.sqlite3',         # Путь к файлу БД
    }
}
```
### **Основные движки БД:**

| Движок         | Для чего              | Простота              |
| -------------- | --------------------- | --------------------- |
| **SQLite**     | Для обучения и тестов | ✅ Очень просто        |
| **PostgreSQL** | Для продакшена        | ⚠️ Сложнее, но мощнее |
| **MySQL**      | Старые проекты        | ⚠️ Средняя сложность  |
